# CineTrack

CineTrack is a minimalist, responsive Movie Watchlist & Review application built to help users manage their favorite movies. This project was developed as part of a specialized training program to demonstrate expertise in modern web development.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **State Management**: React Context API
- **Deployment**: Vercel

## Features

- **Dashboard**: View all your tracked movies in a clean, responsive grid.
- **Movie Management**: Easily add, delete, and toggle "Watched" status for movies[cite: 1].
- **Interactive UI**: Responsive design with hover effects and smooth transitions[cite: 1].
- **Personal Branding**: Integrated professional layout with custom navigation and footer[cite: 1].
