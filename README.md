# CineTrack

Minimalist Movie Watchlist & Review interface.

## Tech Stack

- React.js (Vite)
- Tailwind CSS
- LocalStorage for Data Persistence

## Local Setup

1. Clone the repo: `git clone https://github.com/roky18/CineTrack-clint.git`
2. Install dependencies: `npm install`
3. Run the app: `npm run dev`
