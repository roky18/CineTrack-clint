# CineTrack

CineTrack is a minimalist, responsive Movie Watchlist & Review application built to help users manage their favorite movies. This project was developed as part of a specialized training program to demonstrate expertise in modern web development.

## Tech Stack

- **Frontend**: React.js, Tailwind CSS, Framer Motion
- **State Management**: React Context API
- **Deployment**: Vercel

## Features

- **Dashboard**: View all your tracked movies in a clean, responsive grid.
- **Movie Management**: Easily add, delete, and toggle "Watched" status for movies.
- **Interactive UI**: Responsive design with hover effects and smooth transitions.
- **Personal Branding**: Integrated professional layout with custom navigation and footer.

## Live Deployment

Live Site: [https://cine-track-clint.vercel.app](https://cine-track-clint.vercel.app)

## Local Setup

1. Clone the repository:
   `git clone https://github.com/roky18/CineTrack-clint.git`
2. Install dependencies:
   `npm install`
3. Run the application:
   `npm run dev`
